# Двумерные массивы
 
Элементы квадратной матрицы размерностью 7 на 7 инициализировать целыми случайными числами от 0 до 100 (начальное значение генератора запросить у пользователя).  Вывести исходный массив в виде матрицы. Элементы в строке разделять знаками табуляции.

Найти наибольшее из значений элементов треугольника в верхней части квадратной матрицы между двух диагоналей. Элементы на границах этой части тоже учитываются.
